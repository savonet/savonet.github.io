Class virtual Request_source.unqueued

class virtual unqueued : kind:Frame.content_kind -> name:string -> object .. end
Class unqueued plays the file given by method get_next_file as a request which is ready, i.e. has been resolved. On the top of it we define queued, which manages a queue of files, feed by resolving in an other thread requests given by get_next_request.
